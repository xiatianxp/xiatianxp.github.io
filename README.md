# UUID Generator

A simple, beautiful, and lightweight UUID Generator tool built with pure HTML, CSS, and JavaScript.

Generate secure random UUIDs instantly directly in your browser. No backend, no database, and no data collection.

## ✨ Features

- 🚀 Generate UUID instantly
- 🔒 100% client-side processing
- 📋 One-click copy
- ⚡ Uses native browser Crypto API
- 📱 Fully responsive design
- 🎨 Modern glass-style UI
- 🌐 Works offline
- 🪶 Lightweight single HTML file
- 🚫 No dependencies
- 🚫 No external libraries

## 📸 Preview

A clean and minimal interface for generating UUIDs.

## 🛠 Technology

- HTML5
- CSS3
- Vanilla JavaScript
